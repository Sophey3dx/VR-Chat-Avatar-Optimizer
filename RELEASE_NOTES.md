# Release Notes

## Version 1.1.0 - Outfit Wizard

### New: Outfit Wizard Tool
A complete wizard for setting up clothing toggles and outfit swap systems!

**Single Item Toggle Mode:**
- Create On/Off toggles for individual items
- Automatic PhysBone detection and toggle integration
- Blendshape detection (Shrink_ItemName patterns)
- Expression Parameter, Animator Layer, and Menu created automatically

**Outfit Set Swap Mode:**
- Define multiple complete outfits (groups of items)
- Only one outfit active at a time
- Radial Puppet or Submenu button selection
- PhysBones toggle with each outfit
- Blendshapes animate per outfit

**Access:** `Tools > VRChat Outfit Wizard`

### Version 1.0.18 - Blender Bridge
