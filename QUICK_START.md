# Quick Start - VRChat Avatar Optimizer

## Installation in 3 Steps

### Step 1: Open Unity Package Manager
- Open your VRChat project in Unity (via VCC)
- Go to: `Window > Package Manager`

### Step 2: Add Local Package
- Click the **"+"** button (top-left in Package Manager)
- Select **"Add package from disk..."**
- Navigate to: `C:\Users\david\Desktop\VR CHAT AVATAR OPTIMIZER`
- Select **`package.json`**

### Step 3: Use the Tool
- Go to: `Window > VRChat Avatar Optimizer`
- Follow the Guide tab for instructions

## Alternative: Manual Installation

If the Package Manager method doesn't work:

1. Copy the `VR CHAT AVATAR OPTIMIZER` folder to your project's `Packages/` directory
2. Reload Unity
3. Done!

## Note

The "Add Repository" button in VCC is for remote repositories (URLs), not local packages. Use Unity's Package Manager instead.
